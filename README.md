# 📖 Project Overview

The **Marketing Campaign Database** is a MySQL-based relational database project developed to simulate a real-world digital marketing environment. It manages campaign information, marketing channels, audience demographics, campaign costs, and performance metrics to help analyze marketing effectiveness.

This project demonstrates practical SQL skills including database design, table relationships, joins, aggregate functions, filtering, grouping, subqueries, and business reporting.

# 🗂 Database Schema

The database consists of five related tables:

- 📢 Campaigns
- 📱 Channels
- 👥 Audience
- 🔗 Campaign_Channel
- 📊 Campaign_Performance
